Dep Mac Signing Pending

Time each dep-*-signing-mac14m2 task run spent pending — the gap between when Taskcluster scheduled the run and when a worker started it (started − scheduled). A growing pending time means tasks are waiting longer for a signing worker to pick them up. Data comes from mozdata.fxci.task_runs / task_definitions via STMO (queries refresh every 6 hours).
